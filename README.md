# Threads_hw
Завдання 2

Створіть консольну програму, яка в різних потоках зможе отримати доступ до 2-х файлів. 
Зчитайте з цих файлів вміст і спробуйте записати отриману інформацію в третій файл. 
Читання/запис мають здійснюватися одночасно у кожному з дочірніх потоків. Використовуйте 
блокування потоків для того, щоб досягти коректного запису в кінцевий файл.

Завдання 4

Використовуючи конструкції блокування, модифікуйте останній приклад уроку таким чином, 
щоб отримати можливість послідовної роботи 3-х потоків.
